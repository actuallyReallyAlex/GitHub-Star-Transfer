createMenu

lib/helpers.js

Creates a menu to display to the user.

createMenu(questions: (Array | Object), menuType: String, previousMenu: String)
Parameters
questions ((Array | Object)) An array of questions to ask the user, containing a Question Object, or the Question Object directly.
menuType (String = 'main') A string to indicate which menu should be created. Default is 'main' for Main Menu.
previousMenu (String = 'main') A string to indicate the previous menu the user was on. Mainly used with the 'Exit' menu.
